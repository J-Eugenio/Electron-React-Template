# Template Electron-React

Template do electron com react.

## Installation

Use o instalador de pacotes [Yarn](https://yarnpkg.com/) ou o [NPM](https://www.npmjs.com/)

```bash
yarn install
ou
npm install
```

## Usage

```bash
yarn start
yarn build
```
